Zazil

Descripción del proyecto

En Todas Brillamos, colaboré como Software Engineer desarrollando una aplicación móvil llamada Zazil. Una tienda para dispositivos Android utilizando Kotlin. La app permite a los usuarios comprar productos de manera segura, integrando la API de Stripe para ofrecer múltiples métodos de pago. Además, diseñé y estructuré la base de datos con MySQL Workbench para gestionar inventario y órdenes de compra, y desplegué el backend en la nube utilizando instancias de AWS EC2, asegurando alta disponibilidad y escalabilidad.

Retos enfrentados

Uno de los retos principales fue garantizar una integración fluida con Stripe, que implicó comprender a profundidad su documentación y asegurar la validación segura de transacciones. Adicionalmente, al tratarse de una fundación con recursos limitados, se requería optimizar tanto el rendimiento como los costos del backend. El despliegue en AWS también presentó desafíos relacionados con la configuración de instancias, la seguridad de la API y la automatización del entorno.
